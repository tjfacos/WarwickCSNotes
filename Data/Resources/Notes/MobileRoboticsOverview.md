# Mobile Robotics: Overview

**Warning:** there are no physical robots, and nothing about the actual locomotion of robots. There is a simulation of a robot with sensing strategies and planning a path for the robot, but nothing about how the kinematics of how robots move in the real world.

## Basic Terms

More terms will be introduced later, but here are some basic ones to start with:

- **Actuator**: move-y thing.
- **Sensor**: sense-y thing.
- **Localisation**: *locating* where the robot is in the environment.
- **Mapping**: mapping out the environment.
- **Mobile**: capable of moving in its environment.

## Why Robots? When Robots?

General advantages of using a robot:

- **Increased efficiency.** A robot can repeat the same actions continuously.
- **Doesn't have the same needs as a human.** No physical / environmental requirements like sleep, food, or breaks, e.g. they don't have to go to the toilet, they don't need to sleep, ~~they don't have dreams to escape~~.
- **Increased accuracy.** Some robots have sensors or actuators that exceed human capability (or can sense things humans can't).

## Robot-Human Interactions

Robots sit on a spectrum based on how much human input is directing them:

- **Autonomous**: fully self-directed.
- **Commanded**: a human commands the robot.
- **Remote-controlled**: a human directly controls the robot's movements.

## Robot Intelligence

>[!info]- Automatic vs Autonomy vs Dynamic Autonomy vs Intelligence
> Four related but distinct ideas that get conflated a lot:
>
> - **Automatic**: pre-programmed actions in response to known inputs. For example, a coffee machine automatically dispenses a coffee cup for the user.
> - **Autonomous**: being able to make decisions.
> - **Dynamic Autonomy**: the level of autonomy can *change*. For example, code agents usually operate in a mode where they do everything except where an action may be dangerous, in which case they ask a human for permission to run a command.
> - **Intelligence**: being able to reason and make decisions even for situations the robot wasn't designed for.
